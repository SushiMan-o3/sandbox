# Recipe Vault Proto - https://recipevault-bay.vercel.app/
A small recipe app, made with fastapi and react to keep a hold of all your recipes. hosted frontend and backend on vercel + a postgresql database which is hosted on Neon Postgres. Source code can be found in this repo under /test_proj where claude was used to plan the frontend archicture and claude design was used to design the frontend. 

Backend was planned and writtened by me but claude was used to assist with debugging and helping fix errors. 
Frontend on the other hand was built by claude using React Vite and axios to send http requests to the backend.
